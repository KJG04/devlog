## Keep - 현재 만족하고 있는 부분

- 각 팀원들의 문제 해결 능력이 뛰어나서 크게 막히는 부분이 잘 없었다.
- 다들 적극적인 태도로 참여한 덕분에 프로젝트가 성공적으로 마무리 되었다.

## Problem - 불편하게 느끼는 부분

- `HttpServletRequest`를 `Service`로 직접 넘기는 것은 3 Layer Architecture를 위반것으로 다른 방법을 찾아야 할 것 같다.
- 커스텀 `Exception`을 사용하여 `@ResponseStatus`를 사용시 내부 호출 문제가 일어날 수 있다.
- N+1 문제를 해결하지 못했다.

## Try - Problem에 대한 해결책, 당장 실행 가능한 것

- `ArgumentResolver`를 통해서 매개변수가 `resolver`를 통해서 처리할 수 있는지를 검사하고 이후 jwt토큰을 추출하여 `User`객체를 반환하여 사용한다.
- `@ExceptionHandler`를 사용하여 내부 호출 문제를 해결한다.
- `JPQL`이나 `EntityGraph`를 이용하여 해결한다.
