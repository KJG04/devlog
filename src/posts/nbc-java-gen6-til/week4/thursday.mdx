CRUD에 대해 공부한 내용을 정리해보겠다.

CRUD는 소프트웨어 개발에서 기본적으로 데이터베이스를 다루는 네 가지 작업을 나타내는 약어이다.

## Create

- 새로운 데이터를 생성하는 작업. 데이터베이스에 새로운 레코드를 추가하는 것을 의미.

## Read

- 데이터를 읽어오는 작업. 데이터베이스에서 저장된 정보를 조회하거나 검색하는 기능을 수행.

## Update

- 기존 데이터를 수정하는 작업. 이미 저장된 레코드를 변경하거나 업데이트하는 것을 뜻함.

## Delete

- 데이터를 삭제하는 작업. 데이터베이스에서 불필요한 정보를 제거하는 것을 의미.

이 네 가지 작업은 대부분의 데이터 조작 작업에서 기본적으로 사용되며, 웹 애플리케이션의 백엔드 개발에 중요한 역할을 한다.
