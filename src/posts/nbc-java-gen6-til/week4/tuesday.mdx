GROUP BY는 MySQL에서 동일한 값들을 가진 행들을 그룹으로 묶어 요약 정보를 제공하는 SQL 절이다. 주로 집계 함수(COUNT, MAX, MIN, SUM, AVG)와 함께 사용된다.

```sql
SELECT column1, AVG(column2)
FROM table_name
GROUP BY column1
```

## 집계 함수

- COUNT(column): 특정 열의 값 개수.
- SUM(column): 특정 열의 값들을 모두 더함.
- AVG(column): 특정 열의 값들의 평균.
- MAX(column): 특정 열의 값들 중 최대값.
- MIN(column): 특정 열의 값들 중 최소값.
