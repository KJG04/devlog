react-pixi는 React와 Pixi.js를 결합하여 선언적인 방식으로 Pixi.js를 사용할 수 있게 해주는 라이브러리입니다. pixi-spine은 Spine 애니메이션을 Pixi.js 렌더링 엔진과 함께 사용할 수 있게 해주는 라이브러리입니다.

## 설치

CRA를 사용해 React 18 앱을 생성하고, `pixi.js`, `@pixi/react`, `pixi-spine`를 설치합니다.

```sh
yarn add pixi.js @pixi/react pixi-spine
```
